# junit-reporter

## Install

```bash
go install github.com/bavix/junit-reporter@latest
```

#### Alternative installation

```bash
cd /tmp
git clone https://github.com/bavix/junit-reporter.git
cd junit-reporter
go install -ldflags "-s -w"
```

## Usage

```bash
junit-reporter
```
